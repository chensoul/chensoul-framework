- chensoul-dependencies
- chensoul-framework

### Prerequisites

- Jdk 17+
- Maven 3.2.5

### How to?

```bash
# Clone the repository
$ git clone https://github.com/chensoul/chensoul-framework.git
cd chensoul-framework

./mvnw install -Dgpg.skip=true
```
